# Day 1 - Cloud Computing Basics
## What is Cloud Computing?
Cloud Computing means computing resources(servers, storage, databases, networking, software etc.) over the internet instead of relying only on your own computer.
## Example:
-  **Google drive** stores your file in the cloud.
- **Netflix** streams movies from cloud servers.
- **Gmail** stores your emails in the cloud.
## Types of Cloud
- **Public Cloud:** Shared cloud services provided over the internet(e.g., AWS, Microsoft Azure, Google Cloud).
- **Private Cloud:** Cloud infrastructure used by a single organization.
- **Hybrid Cloud:** A combination of public and private cloud environments.
## Benefits of Cloud
- Lower cost
- Easy to scale up or down
- High availability
- Better security feature
- Access from anywhere with an internet connection
