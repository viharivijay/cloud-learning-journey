
#  Cloud Deployment Models

## 1. Public Cloud
- Shared, internet-based
- Low cost, scalable
- Examples: AWS, Azure, GCP

## 2. Private Cloud
- Dedicated to one org
- Secure, controlled, costly
- Examples: Banks, Gov, Enterprises

## 3. Hybrid Cloud
- Mix of public + private
- Flexible, secure, complex
- Example: Data in private, website in public

## 4. Community Cloud
- Shared by similar orgs
- Collaborative, cost-shared
- Examples: Hospitals, Universities, Gov departments

---

##  Quick Recap
- **Public:** Shared, cheap, scalable  
- **Private:** Secure, controlled, expensive  
- **Hybrid:** Flexible mix, complex  
- **Community:** Shared by orgs with common needs
