### Key Learnings

🔹 Virtualization

- Enables multiple virtual machines to run on a single physical server.
- Improves hardware utilization and reduces infrastructure costs.
- Forms the foundation of modern cloud computing.

🔹 Hypervisors

Hypervisors create and manage virtual machines.

Types:

- Type 1 (Bare Metal): Runs directly on physical hardware.
- Type 2 (Hosted): Runs on top of an existing operating system.

🔹 Virtual Machines (VMs)

- Software-based computers with their own operating system.
- Provide strong isolation between applications.
- Suitable for running different operating systems on the same hardware.

🔹 Containers

- Lightweight environments that package applications with their dependencies.
- Share the host operating system kernel.
- Faster to start and consume fewer resources than virtual machines.

🔹 VM vs Containers

Virtual Machines      | Containers
Own Operating System  | Share Host OS
Higher Resource Usage | Lightweight
Slower Startup        | Faster Startup
Better Isolation      | Better Portability

---

**Key Takeaways**

- Virtualization is the backbone of cloud infrastructure.
- Hypervisors enable efficient management of virtual machines.
- Containers provide faster, portable, and scalable application deployment.
- Understanding the differences between VMs and containers is essential for cloud engineers.
