## Notes

### 1. Virtualization

Virtualization is the technology that allows multiple virtual computers to run on a single physical machine.
It improves hardware utilization, reduces costs, and makes resource management easier.

**Advantages:**
- Better resource utilization
- Reduced infrastructure cost
- Faster deployment
- Easy backup and recovery
- Improved scalability

### 2. Hypervisor A hypervisor is software that creates and manages virtual machines.

**Types:**
Type 1 (Bare Metal): Runs directly on hardware (e.g., VMware ESXi, Microsoft Hyper-V).
Type 2 (Hosted): Runs on top of an operating system (e.g., Oracle VirtualBox, VMware Workstation).

### 3. Virtual Machine (VM)

A VM is a software-based computer with its own operating system, CPU, RAM, and storage.
Multiple VMs can run independently on one physical server.

**Advantages:**
- Strong isolation
- Better security
- Supports different operating systems
- Easy migration and backup

### 4. Containers

Containers package an application with all its dependencies.
They share the host operating system kernel, making them lightweight and fast.

**Benefits:**
- Faster startup
- Lower resource usage
- Easy deployment
- Highly portable
