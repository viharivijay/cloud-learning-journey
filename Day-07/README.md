# Day 7 – Cloud Networking Fundamentals

## Topics Covered
- Cloud Networking
- Virtual Private Cloud (VPC)
- Public and Private Subnets
- IP Addressing
- Internet Gateway
- NAT Gateway
- Route Tables
- Security Groups vs Network ACLs

## Learning Outcome
Today I learned how cloud networks are designed using VPCs, subnets, gateways, route tables, and security controls. I now understand how cloud resources communicate securely within a virtual network and with the internet.

## Key Takeaways
- A VPC provides an isolated virtual network.
- Public subnets allow internet access.
- Private subnets protect sensitive resources.
- Internet Gateway enables public connectivity.
- NAT Gateway allows outbound internet access from private subnets.
- Security Groups and Network ACLs help secure cloud resources.
